Template Strings

É um recurso que possibilita, dentro de uma string, abrir uma janela para o processador JavaScript interpretar a codificação inserida entre chaves ('{}') e interpolar o resultado dessa interpretação ao restante da string.

Ex:Para fazer a interpolação de um texto com variáveis ao invés de usarmos '+' para fazer a interpolação usariamos o código entre crases (` `) e a variável dentro de chaves precedidas por um cifrão (${}).
Interpolação normal: 'O nome iformado é ' + nome + 'e a idade é ' + idade + 'anos.'
Template String = `O nome informado é ${nome} e a idade é ${idade} anos. `

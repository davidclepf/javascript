Atividade de Fixação - Condicional

1) Crie uma aplicação capaz de identificar a faixa etária com base na idade informada pelo usuário. Considere os seguintes critérios:

  1. Se a idade informada for maior ou igual a 0 e menor que 15, exibir a mensagem “Criança”.
  2. Se a idade informada for maior ou igual a 15 e menor que 30, exibir a mensagem “Jovem”.
  3. Se a idade informada for maior ou igual a 30 e menor que 60, exibir a mensagem “Adulto”.
  4. Se a idade informada for maior ou igual a 60, exibir a mensagem “Idoso”.
Fique à vontade para utilizar qualquer uma das funções aprendidas para exibição de dados para o usuário.

DOM - Selecionando Elementos



getElementById()

Seleciona um elemento apartir de um determinado ID.
Ex: console.log(document.getElementById('nome_usuario'))

getElementsByTagName()

Seleciona elementos apartir do nome de uma determinada tag. Serão selecionados todos os elementos pertencentes à tag informada.
Ex: console.log(document.getElementsByTagName('input'))

getElementsByClassName()

Seleciona elementos apartir de uma determinada classe. Serão selecionados todos os elementos pertencentes à classe informada.
Ex: console.log(document.getElementsByClassName('campo_texto'))

getElementsByName()

Seleciona elementos apartir de um determinado nome. Serão selecionados todos os elementos que possuirem o nome informada.
Ex: console.log(document.getElementsByName('nome'))
